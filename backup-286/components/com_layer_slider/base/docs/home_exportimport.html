Exporting sliders

You can export your sliders for backup purposes or to move them between different Joomla! installations. You can export all of your sliders in a single step, as well as selecting them individually. Your browser will download an export file containing your sliders once you hit the "Export" button. Keep this file at a secure location.

Importing sliders

Choose the export file you previously downloaded and click the "Import" button. This will bring back your previously exported sliders immediately.

If you have an export code from an earlier version you just need to paste it into a new file and upload it in the same way. The name of the file could be anything, it does not have any significance, but you need to use the extension ".json".

What about my images?

LayerSlider also export the images used in your sliders. The images also can be found in the downloaded zip package.
